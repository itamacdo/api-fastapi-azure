# Deploy de API FastAPI na Nuvem Azure

Este projeto demonstra como criar e publicar uma API Python utilizando o framework **FastAPI** e o serviço **Azure App Service**. O objetivo é aplicar na prática os conceitos de computação em nuvem e deploy contínuo.

## Funcionalidades

- Endpoint principal de boas-vindas.
- Verificação de status da API.
- Configuração pronta para ambiente de produção (Gunicorn/Uvicorn).

## Tecnologias Utilizadas

- **Python 3.x**
- **FastAPI** (Framework web moderno e rápido)
- **Azure App Service** (Hospedagem PaaS)
- **Git** (Controle de versão)
